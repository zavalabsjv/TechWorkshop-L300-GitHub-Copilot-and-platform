# Task 02 - Fork the repository

## Description

During this task you will fork the repository for this lab into a new repostory within your GitHub Enterprise organization. 

## Success Criteria

- You have forked the repository so that you can make changes during later exercises

## Learning Resources

## Key Tasks

- Navigate to the repository holding the lab materials
- Fork the lab repository

### 01: Navigate to the lab materials repository

Make sure you are logged in to your GitHub account that is connected to the GitHub Enterprise enabled organization.

Navigate to [this GitHub repository](https://github.com/zava-ai-lab/zava-ai-devops-copilot). 

### 02: Fork the lab repository

You will need to fork the repository in order to associate the repository with your GitHub Enterprise account and settings that will apply to the repository. 

Fork [this GitHub repository](https://github.com/zava-ai-lab/zava-ai-devops-copilot) to your account. 

<details markdown="block">
<summary><strong>Expand this section to view the steps</strong></summary>

1. From the lab repository select the **Fork** button.

![Fork the GitHub repository](/media/Solution/0102_fork.png)

2. On the next page, select your account as the Owner and leave the repository name alone. Then, select **Create fork** to complete the process.

![Complete the process of forking the GitHub repository](/media/Solution/0102_fork2.png)
