# Task 01 - Enable GitHub Enterprise

## Description

During this task you will set up GitHub Enterprise for your organization. In a later exercise we will configure GitHub Advanced Security (GHAS).

## Success Criteria

- You have configured your GitHub Enterprise account and connected it with your GitHub organization

## Learning Resources

- [Setting up GitHub Enterprise](https://github.com/account/enterprises/new)

## Key Tasks

- Login to your GitHub account
- Set up a GitHub Enterprise account and connect it to your organization

### 01: Login to your GitHub account

Before you setup GitHub Enterprise, choose the GitHub account you would like to connect to GitHub Enterprise and login to that account.

### 02: Enable GitHub Enterprise 

If you will not be using an organization in your GitHub account that already has GitHub Enterprise, you should choose an organization for which you have authority to add GitHub Enterprise. 

Sign up for a GitHub Enterprise account and associate the organization. 

> **NOTE** You will be asked to provide billing information for the GitHub Enterprise 30 day trial during this lab, however, you will be able to remove this setup during the resource cleanup exercise. 

<details markdown="block">
<summary><strong>Expand this section to view the steps to complete</strong></summary>

1. Sign up for a GitHub Enterprise account

Navigate to: https://github.com/account/enterprises/new to arrive to the signup for GitHub Enterprise. 

![GitHub Enterprise Sign Up Start](/media/Solution/0101_GHE_signup1.png)

If you are not working in a corporate GitHub account, choose "Get started with personal accounts". 

2. Create your enterprise

Enter the name of your enterprise and choose the GitHub organization to connect to. Fill out the remaining details on the form and select "Create enterprise". 

![GitHub Enterprise Sign Up Details](/media/Solution/0101_GHE_signup2.png)

3. Navigate to your enterprise at 
`https://github.com/enterprises/<your enterprise name>` and select "Billing and licenseing" from the menu. Fill out your billing details for the 30 day trial in order to enable the features necessary for this lab. 

</details>
